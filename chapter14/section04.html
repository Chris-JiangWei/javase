枚举的特点

枚举本质上是一个语法糖(指计算机语言中添加的某种语法，这种语法对语言的功能并没有影响，但是更方便程序员使用，同时提高代码的可读性)，可以理解为对JDK1.5以前自定义枚举语法的简写。

特点:

  1. 所有的枚举都继承自java.lang.Enum类，由于Java 不支持多继承，所以枚举对象不能再继承其他类。
  2. 枚举类的所有实例对象都必须放在第一行展示，并且默认都是以public static final修饰的常量，所以变量名通常都全大写。
  3. 在创建实例对象时，不需使用new 关键字。如果调用的是空参数的构造方法，小括号也可以省略。
  4. 使用enum定义、非抽象的枚举类默认使用final修饰，不可以被继承。
  5. 枚举类的构造器只能是私有的，不允许在外部创建对象。
public enum Week {
    // 枚举的第一行必须是创建的实例对象
    // 在创建枚举对象时，不需要使用关键字 new,如果调用的是空参构造方法，小括号也可以省略
    SUN, MON, TUE("星期二"), WED, THU, FRI("星期五"), SAT("星期六");

    Week() {
    }

    // 和普通类一样，枚举里也可以自定义构造方法
    private String name;
    Week(String name) {
        this.name = name;
    }
}
