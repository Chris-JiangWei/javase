二维数组

  • 二维数组：本质上就是元素为一维数组的一个数组。

  • 二维数组的标记：[][]

int[][] arr; //arr是一个二维数组，可以看成元素是int[]一维数组类型的一个数组

二维数组也可以看成一个二维表，行*列组成的二维表，只不过这个二维表，每一行的列数还可能不同。但是每一个单元格中的元素的数据类型是一致的，例如：都是int，都是String等

二维数组的声明与初始化

语法格式：

//推荐
元素的数据类型[][] 二维数组的名称;

//不推荐
元素的数据类型  二维数组名[][];
//不推荐
元素的数据类型[]  二维数组名[];

面试：

int[] x, y[];
//x是一维数组，y是二维数组

动态初始化

二维数组的动态初始化包含两种方式:

  • 指定每一个维度的长度，然后再一个个的填充元素。

    // 生成的是一个3行5列的矩阵
int[][] arr = new int[3][5];

// 动态初始化后，不能直接给二维数组赋值，只能使用下标来给元素赋值
// arr = {{3, 8, 9, 5, 1}, {4, 2, 7, 3, 6}, {9, 6, 7, 4, 3}};

// 使用下标填充数组里的每一个元素
arr[0][0] = 3;
arr[0][1] = 8;
arr[0][2] = 9;
arr[0][3] = 5;
arr[0][4] = 1;

arr[1][0] = 4;
arr[1][1] = 2;
arr[1][2] = 7;
arr[1][3] = 3;
arr[1][4] = 6;

arr[2][0] = 9;
arr[2][1] = 6;
arr[2][2] = 7;
arr[2][3] = 4;
arr[2][4] = 3;

  • 指定第一个维度的长度，再在一维数组里填入数组。

    // 创建一个二维数组，只指定第一个维度的长度
int[][] arr = new int[3][];

// 通过下标，给每个维度填充数据。注意，填充的值也是一个数组。
arr[0] = new int[]{3,2,1,5};
arr[1] = new int[]{7,4};
arr[2] = new int[]{6,8,9};

    思考:能不能不给第一个维度指定长度，而给第二个维度指定长度，即int[][] arr = new int[][4];

静态初始化

元素的数据类型[][] 二维数组名 = new 元素的数据类型[][]{
            {元素1，元素2，元素3 。。。}, 
            {第二行的值列表},
            ...
            {第n行的值列表}
        };

元素的数据类型[][] 二维数组名;
二维数组名 = new 元素的数据类型[][]{
            {元素1，元素2，元素3 。。。}, 
            {第二行的值列表},
            ...
            {第n行的值列表}
        };

//以下格式要求声明与静态初始化必须一起完成
元素的数据类型[][] 二维数组的名称 = {
            {元素1，元素2，元素3 。。。}, 
            {第二行的值列表},
            ...
            {第n行的值列表}
        };

如果是静态初始化，不能手动的指定数组的长度！

举例：

public class Array2Demo1 {
    public static void main(String[] args) {
        // 声明时就直接给数组里的数据元素赋值
        int[][] arr1 = new int[][]{{1,2,3},{4,5},{6,7,8},{9}};

        // 先声明数组，再使用 关键字new 赋值
        int[][] arr2;
        arr2 = new int[][]{{1,2,3},{4,5},{6,7,8},{9}};

        //错误，指定内容以后就不能再指定长度!
        arr2 = new int[3][3]{{1,2,3},{4,5},{6,7,8},{9}};

        int[][] arr3 = {{1,2,3},{4,5},{6,7,8},{9}};;//声明与初始化必须在一句完成
    }
}

杨辉三角是二项式系数的几何表示方式,它没一行的开始和最后一个数字都是1.
从第3行开始,除了每行开始和结束的数字以外,第i行第j个的值是  arr[i][j] = arr[i-1][j-1] + arr[i-1][j]

public static void main(String []args){
        int[][] arr = new int[10][];
        for (int i = 0; i < 10; i++) {
            arr[i] = new int[i + 1];
            arr[i][0] = arr[i][i] = 1;
            if (i > 1) {
                for (int j = 1; j < arr[i].length - 1; j++) {
                    arr[i][j] = arr[i - 1][j - 1] + arr[i - 1][j];
                }
            }
        }

        for (int i = 0; i < arr.length; i++) {
            for (int j = 0; j< arr[i].length; j++) {
                System.out.print(arr[i][j] + " ");
            }
            System.out.println();
        }
    }
