正则表达式

正则表达式是一个特殊的字符序列，计算机科学的一个概念。通常被用来检索、替换那些符合某个模式(规则)的文本。

特点：

  1. 灵活性、逻辑性和功能性非常强；
  2. 可以迅速地用极简单的方式达到字符串的复杂控制。
  3. 对于刚接触的人来说，比较晦涩难懂。

正则规则

  1. 字母和数字表示他们自身，一个正则表达式模式中的字母和数字匹配同样的字符串。
  2. 反斜杠\表示转义字符，表示修改后面紧跟字符的含义。
  3. 多数字母前加一个反斜杠\时会拥有不同的含义。
  4. 标点符号通常都会有特殊含义，如果想要表示它本身，需要在前面加转义字符\.

由于正则表达式是写在字符串里，而在字符串里反斜杠\又是转义字符。所以写在字符串里的正则表达式，如果想要使用转义字符，需要使用两个反斜杠\\.

public class Demo {
    public static void main(String[] args) {
        // 字母表示它本身
        String s1 = "hello".replaceFirst("h", "t");
        System.out.println(s1);  // tello

        //  \ 表示转义字符，如果想要在正则中匹配 \\,需要使用 \\\\
        String s2 = "good\\nok".replaceFirst("\\\\n", "p");
        System.out.println(s2);  // goodpok

        // 字母的前面添加 \\ 会有特殊含义。
        String s3 = "world".replaceFirst("\\d", "x");
        System.out.println(s3); // world

        // 报错，标点符号有特殊含义
        // String s4 = "1+2".replaceFirst("+", "_");

        // 如果想让标点符号表示它本身，需要添加 \\ 注意字符
        String s4 = "1+2".replaceFirst("\\+", "_");
        System.out.println(s4);  // 1_2
    }
}

字母加转义字符的特殊含义

转义后的字母 含义
\\d 匹配一个数字字符
\\D 匹配一个非数字字符
\\w 匹配包括下划线的任何单词字符
\\W 匹配任何非单词字符
\\n 匹配一个换行符
\\r 匹配一个回车符
\\s 匹配任何空白字符，包括空格、制表符、换页符等等
\\S 匹配任何非空白字符

标点符号的特殊含义

标点符号 特殊含义
( ) 标记一个子表达式的开始和结束位置。子表达式可以获取供以后使用
| 指明两项之间的一个选择。
. 匹配除换行符 \n 之外的任何单字符
[ 标记一个中括号表达式的开始,通常用来表示单个字符的取值范围
[0-9] 匹配任何数字。等价于 \d
{ 标记限定符表达式的开始
{n} n 是一个非负整数。匹配确定的 n 次。例如，'o{2}' 不能匹配 "Bob" 中的 'o'，但是能匹配 "food" 中的两个 o。
{n,} n 是一个非负整数。至少匹配n 次。例如，'o{2,}' 不能匹配 "Bob" 中的 'o'，但能匹配 "foooood" 中的所有 o。'o{1,}' 等价于 'o+'。'o{0,}' 则等价于 'o*'。
{n,m} m 和 n 均为非负整数，其中n <= m。最少匹配 n 次且最多匹配 m 次。例如，"o{1,3}" 将匹配 "fooooood" 中的前三个 o。'o{0,1}' 等价于 'o?'。请注意在逗号和两个数之间不能有空格。
* 匹配前面的子表达式零次或多次。例如，zo 能匹配 "z" 以及 "zoo"。 等价于{0,}。
+ 匹配前面的子表达式一次或多次。例如，'zo+' 能匹配 "zo" 以及 "zoo"，但不能匹配 "z"。+ 等价于 {1,}。
? 匹配前面的子表达式零次或一次。例如，"do(es)?" 可以匹配 "do" 、 "does" 中的 "does" 、 "doxy" 中的 "do" 。? 等价于 {0,1}。
^ 匹配输入字符串的开始位置，例如：^h匹配以h开头；在方括号表达式中时，它表示不接受该字符集合，例如[^0-9]匹配除了数字以外的数据
$ 匹配输入字符串的结尾位置。
