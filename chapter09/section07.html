包的使用

所谓的包，本质上其实就是一个文件夹。将多个.java文件放在一个文件夹里，这个文件夹就可以称为包。包的作用:

  • 可以避免类重名：有了包之后，类的全名称就变为：包.类名

  • 控制某些类型或成员的可见范围。如果某个类型或者成员的权限修饰缺省的话，那么就仅限于本包使用。

  • 分类组织管理众多的类，例如：

    • java.lang----包含一些Java语言的核心类，如String、Math、Integer、 System和Thread等，提供常用功能
    • java.net----包含执行与网络相关的操作的类和接口
    • java.io ----包含能提供多种输入/输出功能的类
    • java.util----包含一些实用工具类，如集合框架类、日期时间、数组工具类Arrays，文本扫描仪Scanner，随机值产生工具Random。
    • java.text----包含了一些java格式化相关的类
    • java.sql和javax.sql----包含了java进行JDBC数据库编程的相关类/接口
    • java.awt和java.swing----包含了构成抽象窗口工具集（abstract window toolkits）的多个类，这些类被用来构建和管理应用程序的图形用户界面(GUI)。

声明包名

语法格式:

package 包名;

注意事项:

  1. 包名声明必须要在源文件的代码首行。
  2. 一个源文件只能有一个声明包的语句。
  3. 包名要遵守命名规则与规范:
    • 所有的单词都小写，每个单子之间使用 . 链接
    • 习惯使用公司的域名倒置，例如: com.atguigu.xxx
    • 建议不要使用java.xxx这种格式的包名，避免和Java官方提供的包混淆。

编译和运行声明包的Java代码

package com.atguigu.demo;

public class TestPackage {
    public static void main(String[] args) {
        System.out.println("hello package");
    }
}

编译上述代码:

javac -d . TestPackage.java

-d命令选项 用来指定生成的 class 文件的存储路径

.参数 用来表示当前目录。

-d . 的含义就是说，将生成的 class 文件存放在当前目录。

编译完成以后，会在当前文件夹下生成 com/atguigu/demo 这样的多级目录。

运行class文件

java com.atguigu.demo.TestPackage

运行时需要写完整包名.类名才能找到这个 class 文件。

跨包使用

在开发中，经常需要在一个包里访问另一个包里类的资源，此时就需要跨包来访问。跨包访问的前提是，被访问的资源它的权限修饰符必须是 > 缺省。

通常情况下，有两种方式可以实现跨包访问:

  1. 使用完整名称，即包名.类名(不推荐，写起来很繁琐)

    java.util.Scanner = new java.util.Scanner(System.in);   // 使用完整的包名

  2. 使用import关键字手动导入需要的资源。

    import packageName.className;
import packageName.*;
import static packageName.className.静态成员名;

    例如:

    import java.util.Scanner;
import java.io.*;

Scanner = new Scanner(System.in);  // 直接使用 Scanner类
FileInputStream  / FileOutputStream  // 导入了 java.io.*,可以直接使用java.io包里所有的类

    注意事项:

    • java.lang包里的类不需要使用import语句导入，可以直接使用。
    • import 语句必须要放在 package 语句下，class 语句上。

  3. 包名和import关键字综合使用。如果在不同的包里，有同名的两个类，其中一个可以使用import导入，另一个需要写完整的文件名。

    package com.atguigu.java;

import java.sql.Date;  // import语句需要在 package 语句之下，class语句之上

public class Test {
    public static void main(String[] args) {
        Date d1 = new Date(236023456102L);  // 这里的 Date 类型是 java.sql.Date 类型
        System.out.println(d1);

        // 这里的 Date 使用的是 包名.类名，指的是 java.utli包里的 Date 类
        java.util.Date d2 = new java.util.Date(236023456102L);
        System.out.println(d2);
    }
}
