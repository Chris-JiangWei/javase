变量（variable）

变量：在程序执行的过程中，其值可以发生改变的量

变量的作用：用来存储数据，代表内存的一块存储区域，这块内存中的值是可以改变的。

变量的使用

在使用变量时，必须要指定的三个要素: 数据类型，变量名和变量值。

int a = 10;  // 声明了一个 int 类型的变量，值是 a
int x,y,z;  // 声明了三个 int 类型的变量 x y z
System.out.println(x);  // 报错，变量未初始化，没有值，不能使用

注意：变量的类型在声明时就已经固定了，后续只能存储对应类型的值，不能再存储其他类型(强制转换以后可以，但是可能会出错)。

int a = 1;
a = "hello";  // 报错!!! a是int类型，只能保存整数。

注意事项

  1. 先声明后使用。如果没有声明，会报“找不到符号”错误

  2. 在使用之前必须初始化。如果没有初始化，会报“未初始化”错误

  3. 变量有作用域。如果超过作用域，也会报“找不到符号”错误
  4. 在同一个作用域中不能重名。

标识符( identifier)

简单的说，凡是程序员自己命名的部分都可以称为标识符。即给类、变量、方法、包等命名的字符序列，称为标识符。

标识符在定义时，要遵守一定的规则和规范:

  • 标识符的命名规则（必须遵守）

    • 由字母、数字、下划线以及$组成，不能以数字开头。
    • 不能使用Java的关键字保留字和特殊值
    • 严格区分大小写

  • 标识符的命名规范

    • 命名要做到顾名思义(不要使用拼音和汉字)
    • 变量、方法名等遵守驼峰命名法，即第一个单词首字母小写，以后每个单词首字母都大写，例如:age,userName

    • 类名、接口名、文件名等遵守大驼峰命名法，即每个单词的首字母都大写，例如：HelloWorld，String，System

    • 包名等每一个单词都小写，单词之间使用点.分割，例如：java.lang

    • 常量名等每一个字母都大写，单词之间使用下划线_分割，例如：MAX_VALUE,PI
    • 具有特殊含义的单词按照单词规则写，例如:WTO,iOS

关键字（keyword）

关键字：是指在程序中，Java已经定义好的单词，具有特殊含义。

  • HelloWorld案例中，出现的关键字有 public 、class 、 static 、 void 等，这些单词已经被Java定义好
  • 关键字的特点：全部都是小写字母。
  • 关键字比较多，不需要死记硬背，学到哪里记到哪里即可。

关键字一共50个，其中const和goto是保留字。

true,false,null看起来像关键字，但从技术角度，它们是特殊的布尔值和空值。
